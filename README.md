# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer data to identify churn patterns and retention trends in a subscription-based business.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Key Analysis
- Customer churn rate
- Retention trends
- Contract type analysis
- Customer lifetime value

## Insights
- Month-to-month customers have higher churn rates.
- Long-term contracts improve retention.
- Customers with short tenure are more likely to churn.
